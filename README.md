# DNS 
⚠️***DNS 请使用加密 DNS 来防止 DNS 污染***⚠️

## 国内（国内 DNS 请走直连）
阿里 DOH3

```
h3://223.5.5.5/dns-query
```

阿里 DOT

````tls://223.5.5.5````

腾讯 DOH3

````h3://doh.pub/dns-query````

腾讯 DOT

````dot.pub````


## 国外（国外 DNS 请走代理）

Google DOH3

````h3://dns.google/dns-query````

Google DOT

````tls://dns.google````
